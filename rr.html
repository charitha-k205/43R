IMAGE FUSION USING CONVOLUTION NEUTRAL NETWORKS:

Desprection:-Image fusion is a method of combining source images taken from the same scene. A deep convolutional neural network (CNN) is used in this paper to extract the high frequency details from the two source images. A focus map is generated after the several convolutions and max-pooling layers which contains the clarity information of the source images. A fixed threshold is applied to the focus map to generate a binary segmented map which correctly classifies the pixels belonging to the focused regions.The results of binary segmentation contain some mis-classified pixels which is improved by applying a small region removal strategy to get the initial decision map.
